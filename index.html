El inicio de mi camino

Hacia mi destino...

En mi camino hacia convertirme en un desarrollador, vimos en la CLASE 3 del PrepCourse de HENRY, lo que es la introducción a HTML. Básicamente, HTML se centra en componer simplemente la estructura de una página web. Esto se hace para que se tenga bien "Encuadrada" cada sección de la misma de manera tal que los buscadores, los navegadores, y demás aplicaciones, puedan interpretar correctamente nuestra página para poder trabajarla. Para ello, se utilizan ETIQUETAS. Las etiquetas es el núcleo de HTML podría decirse, para poder dar ese ENCUADRE que estamos buscando a nuestra página web.

Primeras paradas

Etiquetas de formato

Durante esta clase 3, vimos primeramente etiquetas de formato. Las etiquetas de formato, que utilizamos en esta misma página, son las etiquetas denominadas "H" que se utilizan para realizar un titulado. Las mismas van ascendiendo en orden de importancia, siendo H1 la más arriba gerarquicamente:

  1. H1

    2. Notese que el H1 correspondiente es el del título de esta pagina web. Al ser h1 el principal ,acá no está tomando el formato correspondiente ya que debería verse más impactante. Además, no hace bien a nivel CEO y ESTRUCTURA utilizar más de un H1 en nuestra página, pero lo estamos utilizando meramente a modo de utilizar las herramientas para el ejercicio.

  2. H2

  3. H3

  4. H4

  5. H5
  6. H6

Aquí utilizamos la herramienta OL Ordered List / Lista ordenada para enumerar los distintos H. También existe una manera sin ordenar de definir un listado. A modo de ejemplo, listamos los tipos de lista que vimos en la clase:

  • OL/Ordered List/Lista Ordenada
  • UL/UnOrdered List/Lista No Ordenada

Etiquetas de bloques

Luego, vimos las etiquetas de bloques. Las mismas se utilizan para encuadrar secciones que queremos estructurar dentro de la página. Ocupan todo el ancho disponible de nuestra pantalla. Los ejemplos que vimos son:

  • MAIN
  • SECCION
  • ARTICLE

Existe además una especie de ENCUADRE a modo de comodin que es el DIV, el cual utilizamos recurrentemente en el código de esta página.
